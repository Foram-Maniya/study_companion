# 📚 StudySphere AI

An AI-powered Study Companion built to help students learn smarter from their study materials.

## 🚀 Features

- 📄 Upload PDF Notes
- 🤖 AI Chat with Uploaded PDFs
- 📝 AI Quiz Generator
- 📖 AI Summary Generator
- ⚡ Last Minute Revision
- 📊 PYQ Analyzer
- 🔍 Smart Topic Extraction
- 📚 Markdown Answer Rendering

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Supabase
- Supabase Edge Functions

### AI
- Google Gemini API

## 📂 Project Structure

```text
src/
supabase/
public/
```

## ⚙️ Installation

```bash
git clone https://github.com/Foram-Maniya/study_companion.git
cd study_companion
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```

Configure your Gemini API key as a Supabase Edge Function secret.
